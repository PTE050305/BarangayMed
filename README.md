🏥 Barangay Health Center Appointment System

A mobile-based healthcare appointment system built using Android Studio (Java) with a PHP + MySQL backend, designed to streamline patient scheduling, reduce waiting time, and improve healthcare service management at the barangay level.

📱 Features
👤 User Side
🔐 Login & Registration
🏠 Home Dashboard with dynamic Upcoming Appointment
📅 Book Appointment (Service → Date → Time)
📋 View Appointments (Pending, Approved, History)
🔳 Generate QR Code for check-in
🔔 Notifications
👤 Profile Management
🛠 Admin Side
📊 Dashboard with appointment statistics
📋 Manage Appointments:
Approve
Deny
Reschedule
📷 QR Code Scanner for patient check-in
👥 User Management
🧾 Patient Records with history tracking
🔄 Appointment Workflow
User books an appointment:
Select Service
Choose Date & Time
Appointment status:
Pending → Awaiting admin action
Approved → Ready for visit
Completed → Finished appointment
Cancelled → Removed
Admin actions:
Approve / Deny / Reschedule
Approved appointments:
User generates QR Code
Admin scans QR for check-in
